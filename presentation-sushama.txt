What is an object-oriented programming language?
Answer:-
An object-oriented programming language is a programming language that represents the elements of a problem as objects, which contain data and behavior

================
   -Difference between class and object?
class:-
Class is used as a template for declaring and creating the objects.
Object:-
An object is an instance of a class.
